Pertanyaan yang Sering Diajukan (FAQ)

Bagaimana cara saya menerima produknya?

Setelah pembayaran berhasil melalui Lynk.id, link untuk mengunduh file Excel akan secara otomatis dikirimkan ke alamat email yang Anda gunakan saat transaksi.

Apa format file yang akan saya dapatkan?

Anda akan menerima file dalam format `.xlsx` yang kompatibel dengan Microsoft Excel 2013 ke atas dan juga Google Sheets.

Apakah saya bisa meminta kustomisasi?

Saat ini kami hanya menyediakan template siap pakai. Untuk kebutuhan kustomisasi atau jasa pembuatan khusus, silakan hubungi kami melalui kontak yang tertera di profil Lynk.id.

Apakah ada dukungan jika saya mengalami kesulitan?

Tentu saja. Setiap produk disertai dengan panduan penggunaan. Jika masih ada kendala, jangan ragu untuk menghubungi kami.
